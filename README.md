# ALUME | Postres & Snacks Gourmet

Sitio web oficial de ALUME. Dulzura & snacks premium en una experiencia cyberpunk interactiva con pedidos directos a WhatsApp.

## Características
- Fondo animado e interactivo de medusas flotantes.
- Menú responsivo con efecto de cristal esmerilado (glassmorphism).
- Carrito de compras local en ventana modal.
- Formulario de datos con validaciones (WhatsApp a 10 dígitos).
- Envío automático de comandas de pedidos formateadas a WhatsApp.

## Tecnologías Utilizadas
- HTML5 (Estructura semántica)
- CSS3 (Estilos de neón personalizados y diseño responsivo)
- JavaScript (Animación canvas, lógica de carrito e integración de API de WhatsApp)
